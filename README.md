### 2. Power BI Case Study – Personalized baby product offer

-[View live project](https://sebastiananalytics.github.io/powerbi-imiona-personalizacja/)

-[View repository](https://github.com/SebastianAnalytics/powerbi-imiona-personalizacja)

A business case study based on public Polish baby name data.
The project shows how a company selling personalized baby products could decide which names should be included in the main product catalogue and how often the offer should be updated.

**What the project includes:**

* Power BI report design
* Public data ingestion and preparation
* R script for downloading data
* DAX measures for ranking, Top N analysis, year-over-year changes and category logic
* Business interpretation focused on product catalogue decisions
